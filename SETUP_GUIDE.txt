✅ STEP 1: DATABASE UPDATE (REQUIRED)
─────────────────────────────────────────────────────────────────

1. Open phpMyAdmin (http://localhost/phpmyadmin)
2. Select your database: blood_donation_portal
3. Click "SQL" tab
4. Copy and paste this SQL code:

   ALTER TABLE `users` 
   ADD COLUMN `reset_token` VARCHAR(100) NULL DEFAULT NULL AFTER `last_login`,
   ADD COLUMN `reset_token_expiry` DATETIME NULL DEFAULT NULL AFTER `reset_token`;
   
   ALTER TABLE `users`
   ADD INDEX `idx_reset_token` (`reset_token`);

5. Click "Go" button
6. Verify: You should see "2 rows affected" message

  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

Problem: Notification dropdown doesn't open
Solution: Check if Alpine.js is loaded (check browser console)
         Verify JavaScript is enabled

Problem: Password reset token invalid
Solution: Check database - ensure reset_token columns exist
         Token expires after 1 hour - request new one

Problem: Registration redirect still broken
Solution: Clear browser cache
         Verify register.php was updated correctly

Problem: Dropdown shows empty
Solution: Add some test data:
         - Create pending user registrations (for admin)
         - Create blood requests (for donor/hospital)
         - Check database connection
